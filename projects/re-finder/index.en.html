re-finder
  • about

    There are many things that the current web browser still cannot share today. For instance smell, wind, temperature, touch, actual sizes, and many other subjects cannot be shared. This project ultimately aims to explore the browser's possibilities by bringing out and sharing things from within the web which are difficult to share into the physical world.

    Various pictures are scattered throughout the Web. They can be sceneries, people, and many more. In addition, besides what is being taken, each photo is taken from various angles, and each angle symbolizes the photographer's original intentions. However, the viewer can only imagine that angle from the flat image in the browser.
    The re-finder project creates a movable projector by putting a projector and a servo together, in order to show the angle of the camera. For example, a picture of cherry blossom taken by looking up will be displayed from the same angle, as if it is the view from behind the camera. By seeing pictures in the same angle as it was taken, we think that we could improve sharing photographer's intentions and reliving the entire experience of taking pictures

  • implementation
    The hard drive is constructed by 3 servos put together, and a small projector mounted on the tip. Each servo functions to operate the roll, the pitch, or the yaw. The software is constructed by putting a camera and a refinder application together. Because the current exif format is not designed to be compatible with the roll, pitch, and yaw data of the photo's angle, we are currently building a simple iPhone application where the photographer specifies the data of the angle as UserComment. Pictures taken by the camera application can be placed in any web page. The refinder application refers to what is specified in the UserComment in the exif of the photo, giving each servo is a pulse, replicating the angle of the camera.
